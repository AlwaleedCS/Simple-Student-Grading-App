# Student Grade Calculator (C#)

A simple C# console application that calculates student grades based on their score.

## Features
- Input student name and score
- Automatically calculates grade (A, B, C, D, F)
- Simple and beginner-friendly

## How to Run
1. Clone the project
2. Open in Visual Studio
3. Run the project

## Example Output
Student: Ali  
Score: 95  
Grade: A
